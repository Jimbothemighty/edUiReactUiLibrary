# About
Library containing a number of react UI components and theme options

# Theme
Supports:
- Light/ Dark mode
- Theme colours (red, blue, green, yellow) + corresponding text colours

# Components
- Controls (Input, Dropdown, Toggle, Button)
- Table
- Tabs
- Popup
- Context Menu
- Calendar

# Tools
- React
- TypeScript
- Redux
- ESLint
